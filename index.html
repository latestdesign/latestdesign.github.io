Soccer match prediction algorithm

Created a soccer match prediction algorithm from scratch. The 20000+ match dataset was built by regularly scraping the web for soccer match data and odds, and getting the results afterwards. Data was processed using the pandas Python module, and put through a Tensorflow pipeline. Final model was created using bayesian optimization of hyperparameters. Global performance led to a statistical betting edge, at least on the dataset (further testing needed to check for overfitting).

Tensorflow Bayesian optimization Pandas JSON
